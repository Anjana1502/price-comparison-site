**Price Compare – Amazon vs Flipkart**

A simple app to compare book prices between Amazon and Flipkart.

**Features**

Compare prices between Amazon & Flipkart

Works for Books (more categories coming)

Tap any price to buy directly

Shows which one is cheaper

PriceCompare is a lightweight, fast, and no-nonsense price comparison tool for Indian shoppers.
Instantly compare book prices between Amazon India and Flipkart – no clutter, just the best deal.
